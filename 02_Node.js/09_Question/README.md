## Q1. If JS is single threaded how do web workers run in parallel ?

JavaScript itself single threaded, but web workers run in separate provided engine by the browser. Each worker get it's own JS engine instance and execute code independently.

## Q2. Static Vs Dynamic web-site ?

- A static website contains fixed content. 
    
    - Example : Potfolio website, Documentation page etc.

- A dynamic website generated different content for each user or request. It uses backend logic and database.

    - Example : E-commerce website, online banking etc.
